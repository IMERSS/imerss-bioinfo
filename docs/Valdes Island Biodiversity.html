Valdes Island Biodiversity

This is a project of the Valdes Island Conservancy. This biodiversity project has grown out of the citizen science initiatives of the conservancy's annual bioblitzes, the connections of the society to the Royal BC Museum and the UBC Beaty Biodiversity Museum as well as data from local specialists such as Pacific Marine Life Surveys and citizen science data sourced from iNaturalist.

Open source framework created by IMERSS, code available on GitHub
